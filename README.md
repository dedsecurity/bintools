## Installation

Make sure you have installed the dependencies:

  * `python` 3
  * `git`

Clone the [source] with `git`:
 ```sh
   git clone https://github.com/dedsecurity/bintools
   cd bintools
   ```

 [source]: https://github.com/dedsecurity/bintools
 
 # Use
run in the terminal

```bash
python bin 2
```
